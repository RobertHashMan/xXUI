# x-ui
中文|[English](./README_EN.md)  
支持多协议多用户的 xray 面板   
# 安装
在安装前请确保你的系统支持`bash`和`curl`,且系统网络正常  

```
bash <(curl -Ls https://raw.githubusercontent.com/RobertHashMan/xXUI/master/install.sh)
```
## 建议系统

- CentOS 7+
- Ubuntu 16+
- Debian 8+
